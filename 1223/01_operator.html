01자바스크립트 연산자

자바스크립트 연산자는 사용자가 조건식을 만들어서 원하는 결과를 출력하고자 할 때 사용하는 기호

  • 산술연산자 : +, -, *, /, %(나머지)
  • 문자열 결합연산자 : + (문자+숫자, 문자+문자일 경우)
  • 비교연산자 : >, <, >=, <=, ==(값이 같다), ===(값,형식이 모두 같다)
  • 대입연산자 : =, +=, -= ....
  • 증감연산자 : ++(1씩증감), --(1씩감소)
  • 논리연산자 : &&(and), ||(or), !=(not)
  • 삼항조건연산자 : 조건식?참일때 값:거짓일때 값;
  • 타입연산자 : typeof(데이터의 형식을 체크)
