제어문이란? 프로그램의 실행을 제어(control)하기위해 사용하는 실행문으로서,
1.조건문(if, else, if else)
2.선택문(switch case)
3.반복문(while, do while, for)

1.조건문 - if, else, else if

  • if문 : 조건식에 만족하면 실행되는 조건문
  • if else문 : 조건식에 만족하면 true(합격),false(불합격)값이 출력되는 조건문.
  • if else if문 : 조건식이 여러개일 경우 다양한 결과 값을 얻기위한 조건문
