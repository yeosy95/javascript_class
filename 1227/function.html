함수의 정의
함수는 일련의 내용을 삽입하여 사용자가 언제든 호출하여 내용을 실행할 수 있는 실행문
코드의 중복제거 및 재사용이 용이
유지보수에 용이성이 높다
함수의 종류:사용자정의 함수, 내장함수
    함수의 구조 3가지
    1.매개변수가 없는 일반함수
    function 함수명(){
      실행할 내용;
    }
    2.매개변수가 있는 함수
    function 함수명(변수1,변수2...){
      실행할 내용;
    }
    3.리턴값이 있는 함수
    function 함수명(변수1, 변수2, 변수3...){
      실행할 내용;
      return 실행결과;
    }
