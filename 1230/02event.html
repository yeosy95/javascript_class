02event(이벤트)

event : 특별한 상황.

웹페이지에서는 '줌인(아웃)','스와이퍼','프레스','클릭','오버','아웃','드래그','더블클릭','스크롤','키보드'행위를 했을 때 일어나는 사건을 이벤트라고한다.

이벤트의 종류
onclick() - 마우스를 클릭했을 때 이벤트 발생
ondblclick() - 마우스를 더블클릭했을 때 이벤트 발생
onkeydown() - 키보드로 눌렀을 때 이벤트 발생
onkeypress() - 키보드로 눌렀다가 떼었을 때 이벤트 발생
onkeyup() - 키보드를 놓았을 때 이벤트 발생
onmousedown() - 마우스로 바튼을 눌렀을 때 이벤트 발생
onmouseup() - 마우스로 버튼을 놓았을 때 이벤트 발생
onmouseover() - 마우스로 버튼올렸을 때 이벤트 발생
onmouseout() -마우스로 버튼에 올렸다가 떼었을 때 이벤트 발생
window객체의 이벤트 종류
window.onload - 페이지의 로딩이 완료되었을 때 이벤트 발생
abort - 문서가 완전히 로딩되기 전 이미지의 로딩이 중단되었을 때 이벤트 발생
load - 문서의 로딩이 끝나면 이벤트 발생
unload - 페이지가 다른 곳으로 이동할 때 이벤트 발생
resize - 요소에 사이즈가 변경되었을 때 이벤트 발생
scroll - 스크롤바를 움직였을 때 이벤트 발생

이미지에 마우스 오버시 다른 이미지로 대체

이벤트 사용방식 3가지

사용자가 실제 이벤트를 발생시켰을 때 그 이벤트를 대응하여 처리하는 것을 '이벤트 핸들러'라고한다.

이벤트 핸들러는 앞에 'on'을 붙여주고 이벤트에 대한 동작(함수)를 처리

1.인라인 방식으로 이벤트 처리(태그안에)

태그 on이벤트명 = 자바스크립트 코드 예) div onclick="실행할 코드"

이미지 태그에 직접 onclick="this.src"로 작성하여 실행

2.고전방식

객체.on이벤트명 = function(){실행할 코드;}

객체.on이벤트명 = 함수명;function(){실행할 코드;}

3.표준방식

표준방식으로 위 제목을 클릭하면 메시지 창 띄우기

4.실습예제 - 탭메뉴를 클릭시 해당 색상이 적용되게 하기

  • red
  • green
  • blue
이곳에 배경색이 변경됩니다.
